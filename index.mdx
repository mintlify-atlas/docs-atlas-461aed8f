Clash Verge Rev is a continuation of the original Clash Verge project — a fast, lightweight desktop proxy client built with Tauri and React. It bundles the powerful Mihomo (Clash Meta) proxy core and wraps it in a polished, cross-platform interface that supports system proxy, TUN mode, profile management, rule editing, and much more.

## Get started in three steps

<Steps>
  <Step title="Download and install">
    Visit the [Releases page](https://github.com/clash-verge-rev/clash-verge-rev/releases) and download the installer for your operating system. See [Installation](/installation) for platform-specific guidance.
  </Step>
  <Step title="Import a proxy profile">
    Click the **Profiles** tab, then paste a subscription URL or import a local YAML config file. Clash Verge Rev will download and activate the profile automatically.
  </Step>
  <Step title="Enable system proxy or TUN mode">
    Toggle **System Proxy** on the home screen to route browser traffic, or enable **TUN Mode** for system-wide transparent proxying. Your connection is now protected.
  </Step>
</Steps>

<Note>
  Clash Verge Rev uses the Mihomo (Clash Meta) core internally. Any Clash Meta-compatible subscription or configuration file works out of the box.
</Note>
